# DriveEasy – Car Rental Marketplace Platform

DriveEasy is a **peer-to-peer car rental marketplace prototype** where individuals can list their vehicles and others can rent them on a monthly basis.  
This project focuses on designing the **complete frontend workflow and user experience** of a scalable car rental platform.

The goal of this prototype was to design how a real-world rental platform would function, including **user onboarding, vehicle listings, booking flows, and verification processes**.

---

## Tech Stack

- HTML
- CSS
- JavaScript

---

## Features

- User onboarding flow
- Vehicle listing interface
- Car browsing and selection
- Booking workflow
- Verification flow for vehicles and users
- Responsive UI design

---

## Key Highlights

- Designed the **entire frontend workflow independently**
- Focused on **user experience and platform scalability**
- Structured the platform with **trust and safety mechanisms** such as vehicle inspection and user verification
